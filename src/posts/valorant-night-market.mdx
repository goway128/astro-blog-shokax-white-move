夜市（Night Market）是《无畏契约》国服不定期开放的**限时折扣商店**喵，会随机给你几款武器皮肤打折喵。这篇从零开始，一步步带你**下载游戏 → 登录 → 找到并翻开夜市**喵。

<Note type="info" title="先说结论喵">
  夜市不是天天有喵。它由官方**不定期开启**，通常一次持续两周左右喵。没开的时候，游戏里根本看不到入口喵——这很正常，不是你操作错了喵。
</Note>

## 一、下载并安装游戏喵

国服《无畏契约》是 <Underline type="primary">腾讯代理</Underline> 的喵，目前只有 Windows PC 版本喵。安装方式主要有两种喵：**官方安装包**和 **WeGame**喵。

<Tabs defaultValue="official">
  <Tab label="官方安装包" value="official">
    1. 打开浏览器，进入《无畏契约》国服官网喵。
    2. 点击 **下载游戏**，下载官方安装器喵。
    3. 运行安装器，用 **QQ 或微信** 登录你的账号喵。
    4. 选择安装路径，等待客户端把游戏本体下载完喵（约 30GB+，耐心等喵）。
  </Tab>
  <Tab label="WeGame" value="wegame">
    1. 先安装并打开 **WeGame** 客户端喵。
    2. 在搜索框里搜索「无畏契约」喵。
    3. 进入游戏页面，点击 **下载 / 安装**喵。
    4. WeGame 会自动帮你下载并管理更新，之后每次从 WeGame 启动即可喵。
  </Tab>
</Tabs>

<Note type="warning" title="账号与登录喵">
  国服用的是**腾讯账号体系**喵（QQ / 微信登录），和外服的 Riot 账号是两套完全独立的东西喵，皮肤、段位都不互通喵。建议开启登录保护，护好你的皮肤资产喵。

## 二、首次启动与登录喵

安装完成后喵：

1. 从桌面图标或 WeGame 启动 **无畏契约**喵。
2. 用 QQ 或微信扫码登录喵（建议绑定安全手机，保护账号喵）。
3. 首次进入会有一段引导和更新，跟着走完即可喵。
4. 进入主界面后，顶部会看到一排标签喵：<Label type="info">开始游戏</Label> <Label type="info">特工</Label> <Label type="info">藏品</Label> <Label type="info">商店</Label> ……喵

<Note type="primary" title="夜市藏在哪里喵？">
  夜市的入口在顶部的 **商店** 标签里喵，而不是单独的菜单喵。所以第一步永远是先点进商店喵。

## 三、找到并翻开夜市喵

进入 **商店** 后喵：

1. 顶部通常有几个子标签喵：<Text type="orange">精选</Text>、<Text type="green">夜市</Text>、<Text type="blue">特工 / 捆绑包</Text> 等喵。
2. 如果夜市正在开放，你会看到一个 **夜市** 的入口喵，通常带着醒目的紫色调和倒计时喵。
3. 点进去，会看到 **6 张盖着的卡片**喵。
4. 依次点击每张卡片，卡片会**翻开**，露出一款打折的武器皮肤和它的折扣价喵。

<Collapse title="翻牌的规则细节（点击展开喵）" type="primary">
  - 一共 **6 张**卡喵，每张只能翻一次，翻开后不可撤销喵。
  - 每张卡的折扣是**随机**的喵，范围大致在 **10%~49% off**喵。
  - 卡里出现的皮肤来自你**尚未拥有**的皮肤池喵，已买过的不会再出现喵。
  - 想买就点购买喵，用 <Underline variant="wavy" type="warning">VP（Valorant Points）</Underline> 结算喵；不想买可以只翻开看看，不强制购买喵。
</Collapse>

<Note type="success" title="一句话流程喵">
  启动游戏 → 顶部 <Highlight>商店</Highlight> → <Highlight>夜市</Highlight> → 逐张翻开 6 张卡 → 挑折扣满意的下单喵。
</Note>

## 四、常见问题喵

<Collapse title="Q：我在商店里找不到夜市入口喵？">

<Collapse title="Q：翻开的皮肤我不喜欢，能重置吗喵？">
  不能喵。夜市的 6 张卡在**本轮内容是固定的**喵，翻开后不会刷新，也不能重抽喵。下一轮夜市才会重新随机喵。

<Collapse title="Q：折扣价用什么货币？免费的吗喵？">

</Collapse>

<Note type="danger" title="安全提醒喵">
  不要相信任何「代翻夜市」「保证出稀有皮肤」的第三方服务喵，也不要把账号密码给别人喵。夜市完全是官方随机机制喵，没有任何外部手段能改变结果喵。

## 省钱小贴士喵

- 夜市的折扣是随机的喵，**不必每张都买**喵，只挑你真正想要、且折扣够狠（比如 <Text type="green">-40% 以上</Text>）的那张喵。
